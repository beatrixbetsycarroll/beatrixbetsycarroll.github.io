Display Inline vs Inline Block

Display Block

If an element is being displayed block:

  • it will not allow other elements to sit to the left and right of it on the same level as itself by default. Unless you spec This is because there is no line break before or after it (it doesn't need to be on its own line).
  • It will respect only margins/paddings to either side of it, not above or below.
  • it will not respect height and width settings; it will just take on the height and width of what it contains.

Display Inline

If an element is being displayed inline:

  • it will allow other elements to sit to the left and right of it on the same level as itself. This is because there is no line break before or after it (it doesn't need to be on its own line).
  • It will respect only margins/paddings to either side of it, not above or below.
  • it will not respect height and width settings; it will just take on the height and width of what it contains.

Display Inline-Block

If an element is displayed as inline-block:

  • it will allow other elements to sit to the left and right of it on the same level as itself. This is because there is no line break before or after it (it doesn't need to be on its own line...this is an inline characteristic).
  • It will respect only margins/paddings on all sides of it (this is one of its block like characteristics)
  • it will respect height and width settings (this is also one of its block like characteristics)
