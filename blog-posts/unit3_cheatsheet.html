Ruby Hashes, Arrays, Strings, Integers

Hash

  • hash[key]
    • Returns value at key.
    
    Example:
        h = { "a" => 1, "b" => 2 }
    h["a"] #=> returns 1
    
  • hash[key] = value
    • Reassigns value at key.
    
    Example:
        h = { "a" => 1, "b" => 2 }
    h["a"] = 9
    h["a"] #=> returns 9

Array

  • #length
    Returns the number of elements in an array.
    Exmaple: arr = [5,3,1]
             arr.length
             3
  • #index
    Returns the position of the value in an array.
    Takes an element in the array as an argument.
    Exmaple: arr = [5,3,1]
             arr.index(3)
             1
  • #inject
    Returns the sum of all elements in the array.
    Takes two arguments: the starting sum, and a symbol
    Exmaple: arr = [5,3,1]
             arr.inject(0,:+)
             9
